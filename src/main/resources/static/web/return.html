線上購物退貨流程

於網路商店購物消費時，若您選擇非線上付款方式 (例如ATM虛擬帳號付款)， 請於三日內完成付款於本商店購買之商品，皆享有7日鑑賞期之權益(非試用期)。

在您收到商品後7日內若欲退貨，請將商品所有物件(含贈品、零件、說明書、目錄...等)完整收回原出貨箱內， 並至「網頁下方」→「聯絡我們」提出申請。

待客服確認後即完成退貨申請，物流人員將於1-3天內前往原地址收取商品。

※若您在尚未收到商品前已申請退貨，而物流人員仍將商品配送至您的住所，請直接告知物流人員已申請退貨並拒收即可。
